CSS свойство TRANSITION

Без transition C transition

Спецификация

  • transition-property: color;
  • transition-duration: 2s;
  • transition-timing-function: ease;
  • transition-delay: 0;

transition-property

Указывает на свойство, к которому будет применяться анимация.

  • all

  • …
Полный список доступных для анимации свойств

Пример transition-property

Без transition transition-property: background transition-property: all

transition-duration

Промежуток времени, за который произойдет полное изменение от начального значения свойства до конечного значения того же свойства.

Единица измерения:

  • s - секунды (1s, 5s, 0.5s)
  • ms - миллисекунды

Пример transition-duration

Без transition transition-duration: 0.5s
transition-duration: 3s

transition-timing-function

Математическую функция, показывающая, как быстро по времени меняется значение свойства.

  • ease
  • linear
  • ease-in-out
  • ease-in
  • ease-out
  • step-start
  • step-end
  • steps(число, start | end)

  • cubic-bezier(a ,b, c, d)
Официальная спецификация

Пример transition-timing-function

Ease: медленный старт, быстрая середина, медленное окончание

Linear: постоянная скорость

Ease In: медленный старт, быстрое окончание

Ease Out: быстрый старт, медленное окончание

Ease In Out: похоже на ease, но с более выраженным разгоном и торможением

transition-delay

Задает паузу перед началом анимации.

Единица измерения: s - секунды (60s - минута, 0.5s - полсекунды)

Возможно отрицательное значение, в этом случае анимация начнет выполняться не с начала, а с соответствующего данному времени «участка», при этом задержки перед началом не будет.

CSS свойство ANIMATION

CSS свойство ANIMATION

Пример анимации

Loading

Спецификация

  • animation-name: name; (совпадает с именем @keyframes)
  • animation-duration: 1s/1000ms;
  • animation-timing-function: ease;
  • animation-delay: 0s;
  • animation-iteration-count: infinite;
  • animation-direction: reverse;
  • animation-play-state: running;
  • animation-fill-mode: none;

Перед применением animation

Нужно создать @keyframes - кадры анимации по шагам. "Шаги" можно задать с помощью процентов, например:

  1. from / 0% первый шаг анимации
  2. 50% состояние посередине анимации
  3. to / 100% последний шаг

@keyframes анимации вращения:

			@keyframes spin {
			  from {
			    transform: rotate(0deg);
			  }
			  to {
			    transform: rotate(360deg);
			  }
			}

Еще пример анимации и keyframes

			@keyframes around {
			  0% { left: 0; top: 0;}
			  25% { left: 240px; top: 0;}
			  50% { left: 240px; top: 140px;}
			  75% { left: 0; top: 140px;}
			  100%{ left: 0; top: 0;}
			}
			.square{ animation: around 4s linear infinite;}

animation-name

Используется минимум дважды - для @keyframes и для свойства animation.

animation-duration

Определяет длительность одного цикла анимации, и, подобно свойству transition-duration, может быть задано в секундах и милисекундах, и быть целым или дробным.

Пример animation-duration

animation-duration : 2s;
animation-duration : 200ms;
animation-duration : 4.3s;

animation-delay

Подобно transition-delay, можно задавать в милисекундах и секундах, дробным или целым. Определяет задержку перед началом анимации.

По умолчанию 0.

animation-delay: 2s;

animation-iteration-count

Свойство отвечает за число повторений анимации.

По умолчанию – 1, также можно указать любое число(целое или дробное) или ∞.

Примеры animation-iteration-count

animation-iteration-count : 5;
animation-iteration-count : 0.2;
animation-iteration-count : infinite;

animation-direction

Свойство позволяет указать, будет ли анимация воспроизводиться в обратном порядке в каких-либо из циклов.

Значения

normal – все итерации анимации воспроизводятся как обычно.
reverse –все итерации анимации воспроизводятся в обратном порядке относительно того, как они были определены.
alternate – нечетные циклы воспроизводятся в обычном направлении, а четные – в обратном.
alternate-reverse – нечетные циклы воспроизводятся в обратном направлении, а четные – в прямом.

Примеры animation-direction

normal
reverse
alternate
alternate-reverse

animation-play-state

Данное свойство позволяет поставить анимацию на паузу. Для этого нужно задать ему значение paused (по умолчанию и для продолжения анимации – running).

animation-fill-mode

Свойство определяет будет ли анимация воздействовать на элемент вне времени воспроизведения анимации.

Значения animation-fill-mode

None: До и после анимации воздействия на элемент нет

Forwards: Стили сохранятся после окончания анимации

Backwards: Стили анимации будут уже применены до начала анимации

Both: Стили анимации применятся и до, и после анимации

Примеры animation-fill-mode

Красный в состоянии покоя. Становится зеленым в начале анимации. Становится синим после окончания анимации.

Шорткат

animation: name duration timing-function delay iteration-count direction fill-mode play-state;

Пример:
animation: some-name 2s ease-in-out 1s infinite both paused;

Поддержка браузерами

			@ -moz-keyframes name { arguments }
			@ -webkit-keyframes name { arguments; }
			@keyframes name { arguments; }
			.elem {
			  -webkit-animation: arguments;
			  -moz-animation: arguments;
			  animation: arguments;
			}
